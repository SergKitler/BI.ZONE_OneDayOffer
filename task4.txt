def tsum(in_list):
    return in_list[-1] if len(in_list) == 1 else tsum(in_list[:-1]) + in_list[-1]


if __name__ == '__main__':
    print(tsum(list(range(3))))
    print(tsum(list(range(3000))))
    
    
Функция "tsum" представляет собой рекурсивную функцию для поиска суммы всех чисел из заданного списка.
По условию, если поданный на вход список имеет длину больше 1, то суммировать значение последнего элемента в списке с результатом рекурсивно вызванной функции "tsum" с поданным ей на вход списоком без последнего элемента. 
В случае, если поданный на вход функции "tsum" список имеет длину, равную 1, то рекурсия заканчивается, а на выход отдается значение единственного в списке элемента.
Разберем работу функции на примере tsum(list(range(3)))
На вход функции подается список [0, 1, 2], т.е. tsum([0, 1, 2]).
По результатам проверки условия длина списка больше 1, значит суммируем последний элемент списка "2" с результатом рекурсивно вызванной функции tsum([0, 1]).
При вызове функции tsum([0, 1]) по результатам проверки условия, длина больше 1, значит суммируем последний элемент списка "1" с результатом рекурсивно вызванной функции tsum([0]).
При вызове функции tsum([0]) по результатам проверки условия, длина равна 1, значит возвращаем последний элемент, значение которого "0".
Таким образом, получается следующая сумма: 2 + 1 + 0 = 3
